# DOM Design Tool (Figma-like Editor)

A DOM-based design editor that allows users to create and manipulate rectangles and text elements directly on a canvas.

## Features

### Element Creation

- Add rectangle elements
- Add text elements

### Interaction

- Drag elements using mouse
- Resize using corner handles
- Rotate using rotation handle
- Keyboard controls (Arrow keys, Delete)

### Properties Panel

- Width & height control (min/max limits)
- Background color editing
- Text editing
- Text color editing

### Layers Panel

- View all elements as layers
- Select elements from layers
- Bring forward / send backward (z-index control)

### Persistence

- Save project using localStorage
- Load project after refresh

### Export

- Export design as JSON
- Export design as standalone HTML

## Tech Stack

- HTML
- CSS
- Vanilla JavaScript (DOM API)

## How to Run
https://dom-visual-editor.vercel.app/

## Notes

- Entire project is built using DOM manipulation
- No frameworks or libraries are used
